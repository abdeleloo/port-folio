Site Statique — Hugo Yoga Studio

Période : Novembre 2024 – Décembre 2024  |  Contexte : Formation (école) — 1ère année BTS SIO

Premier projet de développement web en BTS SIO. L'objectif était de créer un site statique en HTML et CSS uniquement, sans PHP ni JavaScript. Ce projet a permis de maîtriser les bases de la structure HTML, la mise en page CSS et le responsive design.

1

Aperçu du site

Site vitrine pour un studio de yoga fictif — Hugo Yoga Studio. Le site présente les cours, les horaires et les informations de contact du studio, dans une charte graphique calme et épurée.

Page d'accueil du site Hugo Yoga Studio — structure HTML statique avec navigation, image hero et contenu centré.
2

Structure et contenu

Pages réalisées

  • Accueil — présentation du studio
  • Cours — Hatha, Vinyasa, yoga réparateur
  • Horaire — planning des séances
  • Contact — formulaire statique

Techniques utilisées

  • Structuration sémantique HTML5
  • Mise en page CSS (flexbox, centrage)
  • Charte graphique cohérente (couleurs, typographie)
  • Navigation entre pages statiques

Extrait HTML — Structure de la page d'accueil

<!DOCTYPE html>
<html lang="fr">
<head>
  <meta charset="UTF-8">
  <title>Hugo Yoga Studio</title>
  <link rel="stylesheet" href="style.css">
</head>
<body>
  <header>
    <img src="img/lotus.jpg" alt="Logo Hugo Yoga Studio">
    <h1>HUGO YOGA STUDIO</h1>
    <nav>
      <a href="index.html">Accueil</a>
      <a href="cours.html">Cours</a>
      <a href="horaire.html">Horaire</a>
      <a href="contact.html">Contact</a>
    </nav>
  </header>

  <main>
    <img src="img/salle.jpg" alt="Salle de yoga">
    <h2>Trouvez votre lumière intérieure</h2>
    <p>HUGO Yoga Studio propose tous les niveaux de pratique...</p>
  </main>
</body>
</html>
