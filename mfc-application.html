Application de Gestion MFC

Période : Janvier 2026 – Avril 2026  |  Contexte : Formation (école)

Dans le cadre de la formation BTS SIO, développement d'une application Java de gestion pour l'entreprise fictive MFC. L'objectif était de concevoir une solution complète intégrant une interface utilisateur, une logique métier orientée objet et une base de données SQL.

1

Contexte et besoin

L'entreprise fictive MFC avait besoin d'un outil de gestion permettant de centraliser ses données et d'automatiser certaines tâches de suivi. Le projet a été réalisé en équipe avec une approche orientée objet et une gestion de version via Git.

  • Analyse du besoin et rédaction des spécifications fonctionnelles
  • Modélisation de la base de données (MCD → MLD)
  • Conception des classes Java (diagramme de classes UML)
  • Planification en méthode Kanban (Trello)
Interface principale de l'application de gestion MFC — vue générale de la fenêtre.
2

Conception et architecture

L'application suit un pattern MVC (Modèle-Vue-Contrôleur) pour séparer la logique métier, l'accès aux données et l'interface graphique.

Modèle

Classes Java (POJO), accès BDD via JDBC

Vue

Interface graphique Swing / JavaFX

Contrôleur

Gestion des événements et logique métier

Extrait du code Java dans l'IDE — implémentation de l'architecture MVC de l'application MFC.
3

Développement et fonctionnalités

Fonctionnalités implémentées :

  • Authentification utilisateur (connexion / déconnexion)
  • Gestion CRUD (Créer, Lire, Modifier, Supprimer)
  • Recherche et filtrage des données
  • Connexion à la base de données MySQL via JDBC
  • Export des données
  • Gestion des erreurs et messages utilisateur

Extrait de code – Connexion JDBC

// Connexion à la base de données MySQL
public class DatabaseManager {
    private static final String URL = "jdbc:mysql://localhost:3306/mfc_db";
    private static final String USER = "mfc_user";
    private static final String PASSWORD = "••••••";

    public static Connection getConnection() throws SQLException {
        return DriverManager.getConnection(URL, USER, PASSWORD);
    }
}

// Exemple CRUD : récupérer tous les enregistrements
public List<Client> getAllClients() {
    List<Client> clients = new ArrayList<>();
    String query = "SELECT * FROM clients ORDER BY nom";
    try (Connection conn = DatabaseManager.getConnection();
         PreparedStatement stmt = conn.prepareStatement(query);
         ResultSet rs = stmt.executeQuery()) {
        while (rs.next()) {
            clients.add(new Client(
                rs.getInt("id"),
                rs.getString("nom"),
                rs.getString("email")
            ));
        }
    } catch (SQLException e) {
        e.printStackTrace();
    }
    return clients;
}
