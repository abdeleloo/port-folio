Site Vitrine d'une Salle de Musculation — GYM YONOS

Période : Décembre 2024 – Janvier 2025  |  Contexte : Formation (école) — 1ère année BTS SIO

Deuxième projet web en BTS SIO, cette fois avec l'introduction de PHP. L'objectif était de créer un site vitrine pour une salle de musculation fictive, en ajoutant la gestion des formulaires côté serveur avec PHP.

1

Aperçu du site

Site vitrine pour la salle de musculation fictive GYM YONOS. Le site présente les équipements, les tarifs et un formulaire de contact fonctionnel traité côté serveur en PHP.

Page d'accueil du site GYM YONOS — hero image plein écran, section présentation et galerie photos.
2

Structure et fonctionnalités

Pages réalisées

  • Accueil — hero image + présentation

Extrait PHP — Traitement du formulaire de contact

<?php
// Traitement du formulaire de contact
if ($_SERVER['REQUEST_METHOD'] === 'POST') {

    $nom    = htmlspecialchars(trim($_POST['nom'] ?? ''));
    $email  = htmlspecialchars(trim($_POST['email'] ?? ''));
    $message = htmlspecialchars(trim($_POST['message'] ?? ''));

    // Validation simple
    if (empty($nom) || empty($email) || empty($message)) {
        $erreur = "Tous les champs sont obligatoires.";
    } elseif (!filter_var($email, FILTER_VALIDATE_EMAIL)) {
        $erreur = "Adresse e-mail invalide.";
    } else {
        $succes = "Votre message a bien été envoyé. Merci !";
    }
}
?>
