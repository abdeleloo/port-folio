Développement et intégration de nouveaux modules d'affichage

Période : Avril 2026 – Mai 2026  |  Contexte : Alternance – IP Twins

Dans le cadre de l'évolution de l'application interne d'IP Twins, j'ai développé et intégré de nouveaux modules d'affichage permettant de présenter des données métier de manière plus claire et interactive pour les utilisateurs.

1

Contexte et besoin

L'interface existante présentait certaines données de manière statique, peu lisible, sans possibilité de tri ou de filtrage. La demande était de moderniser ces vues pour améliorer l'expérience des utilisateurs internes.

  • Analyse des besoins avec le responsable technique
  • Inventaire des modules existants à remplacer ou enrichir
  • Conception des nouvelles interfaces (wireframes)
  • Intégration dans l'application existante sans régression

Les captures de l'interface avant intégration ne sont plus disponibles. Le code source et les fonctionnalités décrites documentent l'apport des nouveaux modules.

2

Développement des modules

Chaque module a été développé de façon indépendante puis intégré dans la page existante.

Module Tableau dynamique

  • Tri par colonnes (clic sur en-tête)
  • Filtre par mot-clé en temps réel
  • Pagination côté client
  • Export CSV

Module Statistiques

  • Graphiques interactifs (Chart.js)
  • Données chargées via AJAX (PHP/JSON)
  • Mise à jour dynamique sans rechargement
  • Responsive (mobile/desktop)

Extrait – Tri dynamique d'un tableau

// Tri dynamique d'un tableau HTML au clic sur l'en-tête
document.querySelectorAll('th[data-sort]').forEach(th => {
  th.style.cursor = 'pointer';
  th.addEventListener('click', () => {
    const col = th.dataset.sort;
    const tbody = th.closest('table').querySelector('tbody');
    const rows = Array.from(tbody.querySelectorAll('tr'));
    const asc = th.dataset.order !== 'asc';
    th.dataset.order = asc ? 'asc' : 'desc';

    rows.sort((a, b) => {
      const va = a.cells[col].textContent.trim();
      const vb = b.cells[col].textContent.trim();
      return asc ? va.localeCompare(vb) : vb.localeCompare(va);
    });

    rows.forEach(r => tbody.appendChild(r));
  });
});

// Filtre temps réel
document.getElementById('searchInput').addEventListener('input', function() {
  const q = this.value.toLowerCase();
  document.querySelectorAll('tbody tr').forEach(row => {
    row.style.display = row.textContent.toLowerCase().includes(q) ? '' : 'none';
  });
});
3

Intégration et tests

Les modules ont été intégrés progressivement dans l'application existante, en veillant à ne pas introduire de régression sur les fonctionnalités déjà en place.

  • Tests de non-régression sur les pages modifiées
  • Tests cross-browser (Chrome, Firefox, Edge)
  • Tests responsive (mobile, tablette, desktop)
  • Validation avec le responsable fonctionnel avant mise en production
